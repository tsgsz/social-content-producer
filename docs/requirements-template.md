## 一、目标受众

描述三类主要受众：

### 1. 旅游观众

- 关注点: 出片效果、路线效率、实用信息

### 2. 学生观众
- 特征: 需要学习方法、作业模板
- 需求: 可引用的关键词、句式、分析框架

### 3. 艺术从业者
- 特征: 关注策展逻辑、媒介信息、方法论
- 需求: 策展机制、专业细节、行业观察
- 关注点: 空间叙事、材料运用、跨文化元素

## 三、内容框架

### 微信公众号

**内容类型**: 工具包型 / 短知识型

**字数要求**: 800-1200字

**结构模板**:
```
标题（3个备选）
├── 封面文案
├── 摘要
├── 开头（痛点共鸣）
├── 核心内容（3个方法）
│   ├── 方法1: [具体方法] + [示例]
│   ├── 方法2: [具体方法] + [示例]
│   └── 方法3: [具体方法] + [示例]
├── Checklist（可截图保存）
├── 本周建议
└── CTA + 关键词领取
```

**风格要求**:
- 工具导向，可转发到群
- 提供可下载/打印的工具
- 口语化，避免AI腔调

### 小红书

**内容类型**: 可截图保存的工具型内容

**必含元素**:
- 对比表
- 路线图
- 清单
- 模板
- 机位图

**9图骨架**:
1. 封面：[场景+大字标题]
2. 完整时间表：[小时级安排]
3. 展览路线：[必看标注]
4. 休息区推荐：[实拍场景]
5. 路线图：[步行导航]
6. 日落/拍照点：[具体坐标]
7. 人群适配：[情侣/朋友/独行]
8. 交通指南：[地铁+步行]
9. CTA图：[评论引导]

**写作风格**:
- 结论前置，无铺垫
- 信息密度高
- Emoji适度使用
- 短段落，易扫描

### 抖音

**视频时长**: 60-90秒

**结构模板**:
```
开头 (0-2秒)
├── 钩子字幕
├── 痛点承诺
└── 资料包预告

中段 (3-45秒)
├── 游客党: [看什么] + [为什么]
├── 学生党: [学什么] + [怎么用]
└── 从业者: [看什么] + [专业点]

结尾 (46-60秒)
├── CTA字幕
├── 关键词重复
└── 私信引导
```

**拍摄要求**:
- 真人出镜（开头结尾）
- 展品实拍（中段）
- 快速剪辑
- 字幕加粗重点

## 四、交付标准

### 违禁词检查

**绝对禁用**:
- 最字系列: 最好、最大、最适合、最出片
- 第一系列: 第一、首选、排名第一
- 极限词: 极致、完美、顶级、绝对
- 夸张词: 闭眼跟、必看、错过后悔

**禁用AI腔调**:
- 综上所述
- 本文将介绍
- 首先...其次...最后
- 总而言之

### "去AI化"语言要求

**必须包含**:
- ✅ 第一人称经历: "我就是靠这3个方法"
- ✅ 具体坐标: "往左走200米，看到长椅"
- ✅ 感官描述: "氛围感绝"、"吹吹风"
- ✅ 口语化表达: "踩过的坑"、"挺舒服的"

**必须避免**:
- ❌ 过度总结式语气
- ❌ 机械排比结构
- ❌ 空泛形容词
- ❌ 模板化开场

### Double Check 标准

- [ ] 符合各平台模板结构
- [ ] 每条内容2-5个标题备选
- [ ] 语言自然，像真人写的
- [ ] 包含具体时间和地点
- [ ] 包含价格和感官细节
- [ ] 通过违禁词检查
