# Platform Guidelines Reference

## WeChat Official Account (微信公众号)

### Content Types
1. **Tool-kit Type**: Single-image solution, shareable to groups
2. **Short Knowledge Type**: 800-1200 words, fixed structure

### Structure Template
```
Title (3 alternatives)
├── Cover Copy
├── Summary
├── Hook Opening (pain point resonance)
├── Core Content (3 methods + examples)
│   ├── Method 1: Title + Medium
│   ├── Method 2: Find Repeating Elements
│   └── Method 3: Write Real Feelings
├── Checklist (screenshot-ready)
├── Weekly Recommendation
└── CTA + Keyword Redemption
```

### Writing Style
- Practical and actionable
- Include downloadable/printable tools
- Conversational but informative
- 800-1500 words optimal

### Visual Requirements
- 1 infographic checklist (4:5 or 1:1)
- Cover image (900x383px)

## Xiaohongshu (小红书 / Little Red Book)

### Content Requirements
- Screenshot-worthy, save-worthy tool content
- Every post needs 2+ high-density images:
  - Comparison tables
  - Route maps
  - Checklists
  - Templates
  - Photo spot guides

### Structure Template
```
Title (5 alternatives, emoji-rich)
├── Opening Hook (conclusion first, no铺垫)
├── Schedule/Timeline
├── Exhibition Highlights
├── Cafe/Rest Spots
├── Sunset & Night View
├── Transportation
├── Hashtags (10)
└── CTA (comment for resources)
```

### 9-Image Skeleton
1. Cover (sunset + building + big title)
2. Complete schedule (hourly)
3. Exhibition route + must-see markers
4. Cafe recommendation
5. Walking route map
6. Sunset location map
7. Audience comparison
8. Transportation guide
9. CTA end card

### Writing Style
- First 3 lines = conclusion
- No personal story铺垫
- Information-dense
- Emoji-rich but not cluttered
- Short paragraphs

### Visual Requirements
- All images 9:16 aspect ratio
- Consistent color palette
- Information graphics > photos
- Text-overlay on images acceptable

### Hashtag Strategy
- 2-3 location tags (#香港西九 #M+博物馆)
- 2-3 activity tags (#看展攻略 #周末去哪儿)
- 2-3 audience tags (#情侣约会 #文艺青年)
- 1-2 general tags (#香港旅行 #打卡)

## Douyin (抖音)

### Content Requirements
- Hook in first 2 seconds
- Quick proof with artwork/signage
- Clear action guidance at end
- Max 3 points per video

### Structure Template
```
Title
├── Opening (0-2s): Hook + Promise
├── Middle (3-45s): Audience Segments
│   ├── Tourists: Quick highlights
│   ├── Students: Study value
│   └── Professionals: Curatorial insights
└── Ending (46-60s): CTA with keywords
```

### Shot List per Segment
- Cover shot: 3 exhibitions quick flash + host
- Segment 1: Artwork + explanation + value prop
- Segment 2: (same structure)
- Segment 3: (same structure)
- Ending: Resource package preview + gesture CTA

### Writing Style
- Fast-paced, energetic
- Direct address: "你"
- Benefit-driven
- Keyword repetition for memory

### Visual Requirements
- 16:9 aspect ratio
- Bold subtitles
- Quick cuts
- Text overlays for emphasis
- Host appearance at start and end

### Script Formula
```
"First sentence = Promise/Hook
+ 
"Three segments = [Audience] + [What to see] + [Why]"
+
"Ending = Resource package + Keyword CTA (repeat 2x)"
```

## Universal Requirements

### Target Audiences
Always address 3 audience types:
1. **Tourists**: First-time, time-limited, photo-focused
2. **Students**: Study materials, analysis frameworks
3. **Professionals**: Curatorial insights, methodology

### Forbidden Content
- No exaggeration
- No unverified claims
- No offensive comparisons
- No sensitive topics

### Required Elements
✅ Specific times (10:30-18:30)  
✅ Spatial details (exit C1, walk 10 min)  
✅ Prices (80-120 HKD)  
✅ Sensory descriptions  
✅ Practical tips (last train times)
